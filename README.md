# 💼 Bank Customer Segmentation – Power BI Project

This Power BI project explores customer demographics, transaction behavior, and gender-based financial insights using interactive dashboards. It demonstrates how data storytelling can uncover valuable business insights from transactional data.

---

## 📌 Project Objective

To analyze customer data from a fictional bank and uncover patterns in demographics, financial behavior, and transaction trends. The goal is to support strategic decisions in marketing, customer service, and financial planning.

---

## 📦 Dataset Overview

The dataset includes transactional and demographic details for bank customers. Key columns:

| Column Name           | Description                                      |
|-----------------------|--------------------------------------------------|
| `TransactionID`       | Unique ID for each transaction                   |
| `CustomerID`          | Unique ID for each customer                      |
| `CustomerDOB`         | Date of birth of the customer                    |
| `CustGender`          | Gender of the customer                           |
| `CustLocation`        | City/location of the customer                    |
| `CustAccountBalance`  | Account balance at transaction time              |
| `TransactionDate`     | Date of transaction                              |
| `TransactionTime`     | Time of transaction                              |
| `TransactionAmount`   | Amount transacted (INR)                          |

> ⚠️ Dataset not included due to upload limitations. Please contact the author if needed.
